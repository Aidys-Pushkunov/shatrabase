Shatrabase Grundlagen

Zusammenfassung

Shatrabase is an open source Shatra database GUI. Shatrabase allows you to work with collections of Shatra games in many ways:
browse, edit, add, organize, analyze, etc.

You can also play against an engine, or play two engines against each other.
All time controls can be specified, including Tournament mode.

Shatrabase is cross-platform. It works on Windows and Linux. If anyone had one, Mac OS X too.

Current Features

  • Multi-platform. Supports Windows, Linux and potentially Mac OS X
  • Load and save SGN files
  • Work with multiple databases simultaneously
  • Browse games, including variations
  • Enter moves, variations, comments
  • Setup board, copy/paste SPN
  • Search header (click on columns in GameList header)
  • Display opening tree for current position
  • Analyze and play games using USHI Shatra engines

Development

The code uses the Qt library. To develop, you need Qt 5.0.0 or later. All classes are documented using Doxygen.

Shatrabase is based on open source chess database program ChessX, whose active developers are James Coons and current maintainer Jens Nissen.

Currently, there are two active Shatrabase developers: Stefan Berke and Francis Monkman.

Inactive developers of ChessX include Marius Roets, Sean Estabrooks, Rico Zenklusen, William Hoggarth and Ejner Borgbjerg who wrote significant parts of code. Heinz Hopfgartner made useful tests and maintained Mac port.

If you wish to participate in the project (either as a developer or a tester), please submit a pull request at https://github.com/franzm/shatrabase.
